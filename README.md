# Signal Detection Rover

## Overview
This project involved the development of a Wi-Fi–controlled rover capable of detecting and decoding multiple signal types, including radio, infrared, ultrasonic, and magnetic signals. The system combined hardware signal acquisition with processing and control logic to enable reliable detection and interaction through a web-based interface.

## My Role
I acted as both project leader and a key technical contributor. I coordinated weekly progress, defined milestones, and supported team members to ensure effective system integration. Alongside this, I was responsible for component selection and system-level design decisions under a strict £70 budget.

## Technical Implementation
A core part of my contribution focused on designing the radio signal detection system and enabling efficient signal processing for downstream use.

- Designed a tuned RLC circuit to achieve resonance at a target frequency for signal capture  
- Implemented a signal processing chain including amplification and envelope detection  
- Used a comparator stage to convert analogue signals into stable digital outputs  
- Structured the system to reduce computational load on the rover’s control logic  

This approach ensured that signal detection was handled efficiently at the hardware level, simplifying downstream processing and improving overall system performance.

## System Design Approach
A key aspect of the project was balancing trade-offs between performance, cost, and system complexity. This involved evaluating multiple design approaches (e.g. coil configurations, component choices) and selecting solutions that met both technical and practical constraints.

## Tools & Technologies
- Analogue circuit design (RLC systems, amplification, filtering)
- Signal processing principles
- PCB design (KiCad)
- Hardware prototyping and soldering
- MATLAB (for modelling and validation where applicable)

## Outcome
